Apartment Twin Project

This project is for Digital Twin Project for Apartment.
The project is composed of 4 sub projects.

1. FloorPlanner
   - This project is 2D SVG (Scalable Vector Graphics) floor planner
   - This project is based on homeRoughEditor
   - Floor plan can be drawn with various tools
   - Automatically saves in indexed DB
   - Floor plan can be converted to three.js 3D model
   - This project is intended to be used by admin
<img width="600" height="448" alt="floorplanner" src="https://github.com/user-attachments/assets/a55bd058-4420-45e8-a03e-79c2edf30882" />

2. Editor
   - This project is 3D house editor using Three.js
   - This project is based on Three.js official editor
   - Wall is made from PlaneGeometry
   - Door, Window, Light, Kitchen Sink, Toilet, Bathtub are automatically generated
   - This project is intended to be used by admin to complete basic house model
<img width="600" height="546" alt="editor" src="https://github.com/user-attachments/assets/4cc54fdb-64c3-4515-bfce-997389c723b6" />
  
3. Interior
   - This project is 3D home interior editor using Three.js
   - This project is based on Three.js official editor
   - Various furnitures and appliances can be added
   - This project is intended to be used by individuals to make their own homes
<img width="600" height="547" alt="interior" src="https://github.com/user-attachments/assets/7eb07db0-6761-4df3-81eb-94ab65f0aaee" />

4. Player
   - This project is 3D home simulator/digital twin using Three.js
   - Furnitures and applicances are Cannon bodies
   - Default is FPS (First Person Shooter) style control (WASD + Mouse lookup)
   - Can be connected to Home Assistant program for realtime-update
